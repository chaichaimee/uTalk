author: chai chaimee

Description

uTalk is an NVDA add-on that provides customizable spoken feedback for common clipboard and file operations.


Use standard keyboard shortcuts (Ctrl+C, Ctrl+V, etc.). uTalk will speak the action performed.


This add-on is released under the GPL license.


Hot Keys

NVDA+Alt+T: Switch between English and Thai speech

Features

• Announces actions such as Copy, Paste, Cut, Save, Select All, Undo
• Supports switching between English and Thai speech with NVDA+Alt+T
• Works in text editors, word processors, and Windows Explorer
• Settings panel to configure spoken messages for:
  • Language
  • Copy
  • Copy as Path
  • Copy File
  • Cut
  • Paste
  • Save
  • Select All
  • Undo
  • Redo
• All settings are saved in a JSON file inside the NVDA user configuration folder and reloaded automatically

Installation

  1. Download the latest release of the add-on
  2. In NVDA, open Tools → Manage Add-ons → Install
  3. Browse to the downloaded file and confirm installation
  4. Restart NVDA to activate uTalk
