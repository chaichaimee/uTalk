## Больше чем обычный отклик
**uTalk** — это не просто инструмент уведомлений, это мост между вашим рабочим процессом и вашим творчеством. Забудьте о стандартных системных звуках.

<br>

## Горячие клавиши
### NVDA+Alt+T
* **Одинарное нажатие:** мгновенное переключение языка
* **Двойное нажатие:** открыть настройки uTalk

Легко переключайтесь между настроенными языками (например, английским, тайским и т. д.) прямо во время работы.
